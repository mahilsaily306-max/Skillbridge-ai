# 📌 Overview

SkillBridge AI is a full-stack web application that helps job seekers improve their resumes using AI.

The platform compares resumes with job descriptions, identifies missing technical skills, calculates ATS compatibility, and generates personalized learning recommendations to help users become stronger candidates.

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/mahilsaily306-max/skillbridge-ai.git
```

Go inside the project

```bash
cd skillbridge-ai
```

Install frontend dependencies

```bash
cd frontend
npm install
```

Install backend dependencies

```bash
cd ../backend
npm install
```

Start Backend

```bash
npm run dev
```

Start Frontend

```bash
cd ../frontend
npm run dev
```

Open

```
http://localhost:3000
```
