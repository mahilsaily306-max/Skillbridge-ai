# 🚀 SkillBridge AI

AI-powered Skill Gap Analyzer that compares resumes against job descriptions, identifies missing skills, evaluates ATS compatibility, and recommends personalized learning paths.

## ✨ Features

- Resume Upload (PDF/DOCX)
- AI Resume Analysis
- ATS Score
- Skill Gap Detection
- Resume vs Job Matching
- Learning Roadmap
- Dashboard
- Responsive Design

## 🛠 Tech Stack

### Frontend
- Next.js 16
- React
- TypeScript
- Tailwind CSS v4

### Backend
- Node.js
- Express
- TypeScript

### Database
- JSON Storage
